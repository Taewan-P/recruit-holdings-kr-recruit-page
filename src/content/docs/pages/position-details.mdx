## 데이터 과학자

담당 사업의 전략에 근거해 의사 결정 지원, 매출·이익 극대화가 되는 데이터 활용 제품의 기획·모델 설계·실장 및 개선. 또, 제품에 있어서의 Web 액세스 로그, 앙케이트, 매출, 효과등의 데이터를 이용한 전략의 입안·추진까지를 담당해 드립니다.

키워드: 데이터 계획, 수리 과학, 수리 최적화, 시뮬레이션, 데이터 마이닝

## 데이터 엔지니어

리크루트의 각종 서비스의 데이터를 분석, 활용한 각종 시책을 실시하는 데이터 분석 기반 운영. Google Cloud Platform, Amazon Web Services 등에 구축된 분석 기반의 인프라 운용뿐만 아니라, 데이터 ETL·데이터 파이프라인의 설계·개발이나 Tableau 등 BI 툴의 개발, 신규 인프라·클라우드 기능의 검증·구축·이행도 담당해 드립니다.

키워드: BigQuery · GCP · AWS · NoSQL · 계산 최적화

## 기계 학습 엔지니어

단기·중기 기계 학습 활용 시책의 입안·추진으로부터 기계 학습을 이용한 어플리케이션의 개발·모델 설계·실장. 또, 담당 시스템의 인핸스·운용까지를 담당해 주십니다.

키워드: ML 제품, 알고리즘 설계, MLOps, 기계 학습 공학, 비정형 데이터

## 애널리틱스 엔지니어

데이터에 의한 의사 결정에 빠뜨릴 수 없는 데이터 환경(BI 대시보드나 데이터 마트, 액세스 로그 등)의 설계·개발, 운용 사업·제품에 있어서의 각종 KPI의 설계 지원. 또한, 모니터링 환경의 정비·개선, 클라이언트용 리포트나 업무 지원 솔루션으로서의 BI 대시보드의 개발·운용, 데이터의 품질 기준의 책정과, 모니터링·유지도 담당해 드립니다.

키워드: 비즈니스 인텔리전스, 시각화, 데이터 마트, 데이터 모델링, 데이터 기반, DataOps
